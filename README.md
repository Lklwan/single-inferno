# single-inferno

A personal project inspired by the Korean reality show *Single's Inferno* (单身即地狱).

This repo is used for **practicing Git and GitHub** workflows.

## Quick Start

```bash
# Clone the repo
git clone https://github.com/Lklwan/single-inferno.git
cd single-inferno

# Run the mood tracker
python mood_tracker.py
```

## Git Learning Notes

Here are some Git commands I'm practicing:

```bash
git add <file>          # Stage changes
git commit -m "msg"     # Commit with message
git branch <name>       # Create a new branch
git checkout <name>     # Switch to a branch
git merge <branch>      # Merge a branch
git push origin <branch> # Push to remote
git pull origin <branch> # Pull from remote
```
